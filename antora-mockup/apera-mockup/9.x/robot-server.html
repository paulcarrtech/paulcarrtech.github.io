Choose the Correct Robot Server in Apera Vue

Check which robot server is set up in Apera Vue:

  1. Look at the bottom left corner in the Apera Vue window (highlighted in green below).

The current robot server appears in Apera Vue.

If the current robot server is incorrect for your application:

  1. Select File > Settings > Robot Server.

  2. Select the correct server for the robot you require for your application.
