Robot Calibration Troubleshooting

Before You Begin

Assumptions

  • Apera Vue Computer is activated and running Apera Vue software (version 8.0.0 or higher)

  • VuePort or custom camera pair has already been calibrated

Choose the Correct Robot Calibration Sticker

Apera provides 3 sizes of robot calibration sticker.

Small stickers measure 20 x 20 mm.
Large stickers measure 40 x 40 mm.
Extra-large stickers measure 80 x 80 mm.
If your working distance (vertical distance from the lens of the camera to the bottom of the bin) is above 2 meters, choose the large sticker (40mm x 40mm). This will improve the cameras ability to detect the sticker during robot calibration.

Calibrate the Robot in Its Picking Pose

  1. Set the start/home pose of the robot in the robot calibration program such that J6 is facing down towards the bin. This is similar to the pose at which the robot would be picking.

The calibration sticker must be attached so that it is visible to the cameras when the robot is in this pose.
An example of a robot posed with the sticker visible to the cameras.

Calibrate the Robot with the Sticker Close to the Camera Calibration Plane

When you set the home pose of the robot in the robot calibration program, the sticker must be placed as close as possible:

  • to the plane at which you calibrated the cameras

  • wihtout causing any collisions while the robot moves during calibration
The camera calibration plane should be in the middle or top of your bin. See Camera Calibration Plane Adjustment for more detail.

Increase Robot Sweep Range and Joint Steps

  1. Increase the J4 and J5 joint ranges by about 20 degrees from the default value in the robot calibration program.

This enables a greater deviation up and down from the sticker’s initial image plane at the start pose. This will also enhance the quality of the dataset for better robot calibration.

  1. Increase the number of steps in each of the joint sweeps from the default value in the robot calibration program.

This change allows for more captures during robot calibration.

If you are unsure about making these changes, contact Apera Support. An Apera Robotics Solutions Engineer will assist you.
