Device Activation

Once your Apera Vue Computer is turned on, you’ll see the activation screen shown below.

Apera activation screen

If you don’t see this screen, ensure Vue is running:

  1. Select Menu > Start Apera Vue.

  2. Open a browser window and navigate to localhost:44334.

To proceed from here, you can use the one-time activation link previously sent to you.

If you don’t have an actication link, contact Apera Support to receive one.

  1. Open the activation link in a web browser to see the activation code.

Sample activation code

  1. Type your unique activation code into the Code field on the activation screen.

  2. Select Activate.

Please wait as the latest Apera Vue software is downloaded.

If you need to, you can shutdown the Apera Vue Computer at any time. Loading will resume on the next boot without issue.

If activation fails, try refreshing the activation link page to regenerate the activation code.

Activation codes last for 15 minutes before they must be regenerated.

If you have any other problems with activation, please contact Apera Support.

Once the download is complete, you will see Apera Vue running.

Apera Vue software updated and running.
