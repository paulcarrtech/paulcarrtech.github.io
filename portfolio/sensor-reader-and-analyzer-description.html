TR-DataRX-C™ Attitude and Heading Reference System (AHRS) Sensor Reader and Analyzer

TR-DataRX-C™ is a state-of-the-art sensor reader and analyzer that plugs into our custom RX-UI™ software to read and analyze sensor data. RX-UI™ restores and plots sensor data, interprets data in real time, and provides advanced statistical data analysis. TR-DataRX-C™ can mount to any surface with its included straps. The sensor data transfers by USB or mini-USB cable.

Features

TR-DataRX-C™ provides orientation data and dynamic information in 3D:

  • roll, pitch, and yaw (Euler) angles
  • linear accelerations (3 DOFs)
  • angular accelerations (3 DOFs)
  • angular velocity (3 DOFs)
  • magnetic field (3 DOFs)
  • jerk components (3 DOFs)
  • several key performance indicators (KPIs)

Benefits

TR-DataRX-C™ can record, analyze, and plot real-time data from a variety of sensor units. This makes TR-DataRX-C™ a powerful, multipurpose tool. TR-DataRX-C™ provides crucial data and analysis for applications such as robotics, navigation, control, augmented reality, virtual reality, gaming, and more.

Platform Integration

TR-DataRX-C™ includes our custom RX-UI™ software, which is compatible with Windows, macOS, and Linux. RX-UI™ allows you to perform the following actions:

  • record and store sensor data from the connected TR-DataRX-C™ sensor reader for future use
  • view the performance report for each sensor recording at any time
  • calibrate sensors to provide the most accurate results for your environmental setup
Note
You can export recorded data to common formats such as an Excel workbook or a comma-separated values (CSV) file.

For your convenience, we provide TR-DataRX-C™ with its software development kit (SDK). The SDK is developed in .NET Standard, which is available for all supported OS platforms. The SDK easily connects TR-DataRX-C™ to your desired development environment, such as the Unity game development engine.
