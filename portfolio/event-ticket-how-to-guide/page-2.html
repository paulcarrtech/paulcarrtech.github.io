Do You Need to Build a New Tour?

Some tours can be reused for various events. Here are common examples:

Booking Type Resource Event
%Built City Built City - JWLC 100 Any Built City lecture configured for 100 seats*
%Development Event Development Event Any free development event that requires no pre-registration in Centaman
%Marketing Event Marketing Event Any free Marketing event that requires no pre-registration in Centaman (e.g., media events, industry open houses)
%Membership Event Membership Event Any free Membership event that requires no pre-registration in Centaman (e.g., exhibition openings, AGMs)
%Public Program – Attend By-Donation Public Program – Attend By-Donation Any by-donation public program that requires no pre-registration in Centaman
%Public Program – Attend Free Public Program – Attend Free Any free public program requiring no pre-registration in Centaman

You can reuse tours for other series in the future (e.g., Design Sundays, film screenings, Happy Hours).

You can also reuse tours for Talk & Tour or Talk & Learn events, but you must give these tours exhibition-specific names. This is a request from Visitor Services, so that they can tell in the point-of-sale (POS) interface which exhibition a Talk & Tour or Talk & Learn is for on any given date.

Notes

  • * If another configuration comes into play for Built City, such as an auditorium booking of 209 seats, you must create a new resource and tour button set.
