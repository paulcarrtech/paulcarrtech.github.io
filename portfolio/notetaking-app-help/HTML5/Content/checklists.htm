How to create notes with checklists

You can create checklists to keep track of tasks, goals, and other items. Each list item has a Checkbox that you can check or uncheck as needed.

Creating notes with checklists

Start a note as a checklist or convert an existing note into a checklist.

Starting notes as checklists

You can start a new checklist with the New List button in the Take a Note... bar near the top of Google Keep:

  1. To open a new checklist, click the New List button.
  2. To write a list item, type or paste text in the + List Item field.
  3. To add that list item to the note, press Enter on your keyboard.
  4. If you want to add more list items, repeat steps 2–3 until the checklist is complete.
  5. To save the note, click Close near the bottom-right corner of the note.

Add list items to the top of the checklist with the + List Item field, or add list items below other list items with a blank List Item field. You must insert blank List Item fields into the checklist:

  1. To begin, place the Cursor at the end of the list item text directly above where you want to insert a blank List Item field.
  2. To insert a blank List Item field below the current list item, press Enter on your keyboard.
  3. To write a new list item, type or paste text in the blank List Item field.

Converting text to checklists

You can convert a note's text into a checklist with the Show Checkboxes command accessed through the More button near the bottom edge of that note:

  1. To access the More button
    • when you want to open a note and convert it to a checklist, click on that note.
    • Or
    • when you want to convert a note to a checklist without opening the note, hover the Cursor over that note.
  2. To access the More drop-down menu, click the More button.
  3. To convert the note to a checklist, click Show Checkboxes from the More drop-down menu.

An unbroken line of text will convert into a single list item.

  1. If you want to split the content of a list item into multiple list items
    1. and if you converted the note to a checklist without opening the note in step 1, click on that note.
    2. place the Cursor in the list item text where you want one list item to end and the next to begin.
    3. press Enter on your keyboard to split the list item into 2 list items.
    4. repeat steps B–C until the checklist is complete.
  2. If you opened the note in step 1 or step 4, save that note by clicking Close near the bottom-right corner of the note.

When a note is open, you can rearrange list items with the Drag handle:

  1. To open the note with the checklist that you want to rearrange, click on that note.
  2. To access the Drag handle, hover the Cursor over the list item that you want to move.
  3. To move the list item, click and drag the Drag handle
    • up or down when you want to reorder the checklist.
    • Or
    • right when you want to nest the list item under another list item.
    • Or
    • left when you want to unnest the list item from under another list item.
