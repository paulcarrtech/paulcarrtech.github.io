How to pin notes

You can pin important notes to place them above the rest of your notes. Pinned notes appear in the Pinned section above the Others section.

Pinning notes

Pin a note with the Pin Note button at the top-right corner of that note. You can access the Pin Note button even without editing that note.

Pinning while editing a note

You can pin a note while editing that note:

  1. To access the Pin Note button
    • when you want to pin a new note, click on the Take a Note... bar near the top of Google Keep.
    • Or
    • when you want to pin an existing note, click on that note.
  2. To move the note to the Pinned section, click the Pin Note button.
  3. To save the note, click Close near the bottom-right corner of the note.

Pinning without editing a note

You can pin a note without editing that note:

  1. To access the Pin Note button without editing the note, hover the Cursor over the note.
  2. To move that note to the Pinned section, click the Pin Note button.

Unpinning notes

Unpin a note with the Unpin Note button at the top-right corner of that note. You can access the Unpin Note button even without editing that note.

Unpinning while editing a note

You can unpin a note while editing that note:

  1. To edit the note that you want to unpin, click on that note.
  2. To move that note to the Others section, click the Unpin Note button.
  3. To save the note, click Close near the bottom-right corner of the note.

Archiving a pinned note unpins that note at the same time.

Unpinning without editing a note

You can unpin a note without editing that note:

  1. To access the Unpin Note button without editing the note, hover the Cursor over the note.
  2. To move that note to the Others section, click the Unpin Note button.

Archiving a pinned note unpins that note at the same time.
