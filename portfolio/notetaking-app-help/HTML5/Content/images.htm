How to create notes with images

You can upload or paste image files into notes to save for later.

Your notes that contain image files can also contain text or a checklist.

Uploading image files into notes

Upload image files into new or existing notes.

Uploading image files into new notes

You can upload image files into a new note with the New Note with Image button in the Take a Note... bar near the top of Google Keep:

  1. To open your operating system's File Manager, click on the New Note with Image button.
  2. To indicate which image files will upload, select those files with the Cursor in the File Manager.
  3. To upload your image selection into the note, click the Open button near the bottom-right corner of the File Manager.
  4. To save the note, click Close near the bottom-right corner of the note.

To add more image files to the same note, follow the steps in the next section.

Uploading image files into existing notes

You can upload image files into an existing note with the Add Image button near the bottom edge of that note:

  1. To access the Add Image button
    • when you want to open a note and upload images into it, click on that note.
    • Or
    • when you want to upload images into a note without opening the note, hover the Cursor over that note.
  2. To open your operating system's File Manager, click on the Add Image button near the bottom edge of that note.
  3. To indicate which image files will upload, select those files with the Cursor in the File Manager.
  4. To upload your image selection into the note, click the Open button near the bottom-right corner of the File Manager.
  5. If you want to upload more image files into the note, repeat steps 2–4.
  6. To save the note, click Close near the bottom-right corner of the note.

Pasting image files into notes

You can paste an image file into a note. This image may be copied from your web browser, your file manager, your screenshot tool, or elsewhere:

  1. To prepare, copy an image file from another source.
  2. To open a note
    • when you want to paste the image file into a new note, click on the Take a Note... bar near the top of Google Keep.
    • Or
    • when you want to paste the image file into an existing note, click on that note in Google Keep.
  3. To upload your copied image file into the note, paste that file into the body of the note.
  4. If you want to add more image files to the note, repeat steps 1 and 4.
  5. To save the note, click Close near the bottom-right corner of that note.
